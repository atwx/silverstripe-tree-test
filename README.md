## Silverstripe Tree Test

This is a repository to test the silverstripe cms tree.

To install, just run `ddev start`, it will run `composer install` and `sake db:build`. `silverstripe-populate` will create the test pages. The site will be available under https://silverstripe-tree-test.ddev.site. The error will be visible under https://silverstripe-test.ddev.site/admin/pages/edit/show/55
